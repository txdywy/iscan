**Create `internal/probe/middleware/chain.go`:**

```go
package middleware

import "iscan/internal/probe"

// Middleware is a decorator that wraps a Probe with additional behavior.
type Middleware func(probe.Probe) probe.Probe

// Chain composes middleware outermost-first.
// Chain(probe, a, b, c) executes as a(b(c(probe))).
func Chain(p probe.Probe, mws ...Middleware) probe.Probe {
    for i := len(mws) - 1; i >= 0; i-- {
        p = mws[i](p)
    }
    return p
}
```

**Create `internal/probe/middleware/timeout.go`:**

Extract the `probeContext` logic from `internal/scanner/scanner.go:220-231` into this middleware. The function computes a per-probe timeout capped to the parent context's remaining deadline.

```go
package middleware

import (
    "context"
    "time"

    "iscan/internal/model"
    "iscan/internal/probe"
)

// Timeout returns a Middleware that wraps each probe invocation in a
// context.WithTimeout. The timeout is capped to the parent context's
// remaining deadline, so a single probe cannot exhaust the entire scan window.
func Timeout(timeout time.Duration) Middleware {
    return func(next probe.Probe) probe.Probe {
        return probe.ProbeFunc(func(ctx context.Context, target model.Target) model.ProbeResult {
            pctx, cancel := probeCtx(ctx, timeout)
            defer cancel()
            return next.Run(pctx, target)
        })
    }
}

// probeCtx derives a child context whose deadline is min(parentDeadline, now+timeout).
func probeCtx(ctx context.Context, timeout time.Duration) (context.Context, context.CancelFunc) {
    if deadline, ok := ctx.Deadline(); ok {
        remaining := time.Until(deadline)
        if remaining < timeout {
            timeout = remaining
        }
        if timeout <= 0 {
            timeout = time.Nanosecond
        }
    }
    return context.WithTimeout(ctx, timeout)
}
```

**Create `internal/probe/middleware/retry.go`:**

Reuse the exponential backoff logic from `scanner.retryWithBackoff` (lines 284-319), adapted to the Probe interface. The retry middleware calls next.Run() up to maxAttempts times. It stops early if the result is successful (Data's Success field is true). On context cancellation, it returns the last result (or zero value if no attempt completed).

The success check uses a private `isSuccess` function with a type switch over all 6 observation types (DNS, TCP, TLS, HTTP, QUIC, Trace). This is explicit and requires no model changes.

```go
package middleware

import (
    "context"
    "time"

    "iscan/internal/model"
    "iscan/internal/probe"
)

// Retry returns a Middleware that retries the probe up to maxAttempts times
// with exponential backoff starting at baseDelay. Retries stop early when
// the probe result indicates success (observation.Success == true).
func Retry(maxAttempts int, baseDelay time.Duration) Middleware {
    return func(next probe.Probe) probe.Probe {
        return probe.ProbeFunc(func(ctx context.Context, target model.Target) model.ProbeResult {
            var last model.ProbeResult
            for attempt := 0; attempt < maxAttempts; attempt++ {
                if ctx.Err() != nil {
                    if last.Layer != "" {
                        return last
                    }
                    return model.ProbeResult{Layer: last.Layer}
                }
                last = next.Run(ctx, target)
                if isSuccess(last.Data) {
                    return last
                }
                if attempt < maxAttempts-1 {
                    delay := baseDelay * (1 << attempt)
                    timer := time.NewTimer(delay)
                    select {
                    case <-ctx.Done():
                        timer.Stop()
                        return last
                    case <-timer.C:
                    }
                }
            }
            return last
        })
    }
}

// isSuccess checks whether a probe result's Data payload indicates success.
// Each observation type has a Success bool field.
func isSuccess(data any) bool {
    switch v := data.(type) {
    case model.DNSObservation:
        return v.Success
    case model.TCPObservation:
        return v.Success
    case model.TLSObservation:
        return v.Success
    case model.HTTPObservation:
        return v.Success
    case model.QUICObservation:
        return v.Success
    case model.TraceObservation:
        return v.Success
    }
    return false
}
```

**Create `internal/probe/middleware/logging.go`:**

The logging middleware wraps probe invocation to log layer, target domain, status (ok/fail/skip), and latency. It accepts a `func(string, ...any)` logger (compatible with `log.Printf`, `fmt.Printf`, `testing.T.Logf`).

```go
package middleware

import (
    "context"
    "fmt"
    "time"

    "iscan/internal/model"
    "iscan/internal/probe"
)

// Logging returns a Middleware that logs each probe invocation with
// layer, target domain, status (ok/fail), and latency.
// The logger parameter is compatible with log.Printf, fmt.Printf, etc.
func Logging(logger func(string, ...any)) Middleware {
    return func(next probe.Probe) probe.Probe {
        return probe.ProbeFunc(func(ctx context.Context, target model.Target) model.ProbeResult {
            start := time.Now()
            result := next.Run(ctx, target)
            latency := time.Since(start)
            status := "fail"
            if isSuccess(result.Data) {
                status = "ok"
            }
            if result.Layer == "" {
                status = "skip"
            }
            logger("[probe] %-5s %-20s %s (%v)",
                result.Layer, target.Domain, status, latency.Round(time.Microsecond))
            return result
        })
    }
}

// fmt import needed for logger format — remove if logger handles formatting.

    _ = fmt.Sprintf("") // ensure fmt is used (compiler guard)
}
```
